# Favicon Creation Guide for BingeTV

## Required Favicon Files

The following favicon files are referenced in the HTML but need to be created:

1. favicon.ico (16x16, 32x32 - multi-size ICO file)
2. apple-touch-icon.png (180x180 - for iOS home screen)
3. favicon-32x32.png (32x32 PNG)
4. favicon-16x16.png (16x16 PNG)

## How to Create Favicon Files

### Option 1: Use an Online Favicon Generator
1. Create a logo image (preferably SVG or high-resolution PNG)
2. Use an online tool like:
   - https://favicon.io/
   - https://realfavicongenerator.net/
   - https://favicon-generator.org/

3. Upload your logo and generate all required sizes
4. Download the favicon package
5. Replace the placeholder files below

### Option 2: Create Simple Text-Based Favicons
For development purposes, you can create simple text-based favicons:

1. Create a 32x32 PNG with "BTV" text in red (#8B0000) on black background
2. Create a 16x16 version of the same
3. Create a 180x180 version for iOS
